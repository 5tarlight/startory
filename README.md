# startory
Blog for Developers

## Requirements
- MariaDB
- Node.js
- npm or yarn

## Installation
- Download all dependencies
```
yarn add
```
- or if you use npm
```
npm install
```

- SQL query
```
source "YOUR SQL FILE URL HERE";
```
- Rename src/config/dbsetting.inc.json into dbsetting.json
- Edit src/config/dbsetting.json
